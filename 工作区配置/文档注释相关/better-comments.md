# better-comments



personal usage

# 基本语法

~~~javascript
{
    "tag": "!",
    "color": "#FF2D00",
    "strikethrough": false,
    "underline": false,
    "backgroundColor": "transparent",
    "bold": false,
    "italic": false
  },
~~~

# 默认设计

~~~javascript
"better-comments.tags": [
  {
    "tag": "!",
    "color": "#FF2D00",
  },
  {
    "tag": "?",
    "color": "#3498DB",
  },
  {
    "tag": "//",
    "color": "#474747",
	"strikethrough": true

  },
  {
    "tag": "todo",
    "color": "#FF8C00",
  },
  {
    "tag": "*",
    "color": "#98C379",
  }
]
~~~
